Apple Health stores sleep phases as multiple HKCategoryTypeIdentifierSleepAnalysis interval records, not as one nightly session object.
Each record has:
- type
- value
- startDate
- endDate

Deep sleep is stored as:
HKCategoryValueSleepAnalysisAsleepDeep

To compute deep sleep time:
1. Filter records where type == HKCategoryTypeIdentifierSleepAnalysis
2. Filter those where value == HKCategoryValueSleepAnalysisAsleepDeep
3. Compute duration for each as endDate - startDate
4. Sum all durations

Do not guess deep sleep from total sleep.
If only HKCategoryValueSleepAnalysisAsleepUnspecified exists, then stage-level deep sleep is unavailable in the export.

Find Apple Health records with type=HKCategoryTypeIdentifierSleepAnalysis and value=HKCategoryValueSleepAnalysisAsleepDeep, then sum endDate-startDate.
